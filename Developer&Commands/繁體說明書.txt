--------地圖修改--------
* 所有VS地圖均已解除屋頂限制
* 移除地圖上的固定機槍
* 地圖藥丸
	-醫療箱限制兩顆藥丸但不一定會出現
	-救援區域: 四顆
	-路上: 隨機出現1~2顆數量

* 在地圖上新增許多障礙物、武器、梯子 //參考一代rotoblin2、二代L4D2-The Last Stand、二代L4D2-Comp-Stripper-Rework
* 最終關卡路程符合救援進度
* 在地圖上不起眼、卡圖、非倖存者必經之路、太過資源密集的地方移除一些物資，

--------分數計算--------
分數 = ( AD + HB + PILLS ) x Alive x Map  
AD = 平均距離, 介於 0-100%
HB = 生命加值, Floor(實血/2) + RoundToNearest(虛血/4)
PILLS = 藥分, 保留藥丸每顆+15
Alive = 倖存者存活數
Map = 地圖難度修改
